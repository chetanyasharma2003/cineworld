# CineWorld Client

React + Vite frontend for the CineWorld app.

## Environment

Create `client/.env` from `.env.example`:

```env
VITE_API_URL=http://localhost:8000/api
```

The TMDB token belongs in `server/.env`, not in browser code.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```
